## 使用方法

1. 安装python环境
2. 安装pip
3. 克隆仓库
4. 安装requests库
5. 在Google Chrome打开浏览器，登录https://scut.yuketang.cn/，然后F12，点击Application，然后查看Cookies

6. 运行脚本，输入对应课程编号并回车
